Project Description

This project is an online shop application built using:

- Django REST Framework (backend)
- Angular (frontend)

Users can:
- View products
- Create orders
- Authenticate using token-based authentication

Technologies:
- Python / Django REST Framework
- Angular
- PostgreSQL (optional)
- Postman

How to Run

 Backend

  cd backend
  python manage.py runserver

 Frontend

  cd frontend/online-shop-frontend
  ng serve

API Base URL - http://127.0.0.1:8000/api/
